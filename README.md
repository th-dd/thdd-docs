## 🚀 本地开发

### 环境要求

- Node.js 18+
- pnpm 8+

### 快速开始

```bash
# 克隆仓库
git clone https://github.com/th-dd/thdd-docs.git
cd thdd-docs

# 安装依赖
pnpm install

# 启动开发服务器
pnpm docs:dev

# 构建生产版本
pnpm docs:build

# 预览构建结果
pnpm docs:preview
```

## 📖 技术栈

- [VitePress](https://vitepress.dev/) - 静态站点生成器
- [pnpm](https://pnpm.io/) - 包管理器
- [GitHub Pages](https://pages.github.com/) - 静态托管
