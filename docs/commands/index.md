# AstrBot 指令列表

AstrBot 是一个功能丰富的 QQ 机器人框架，提供了群管理、娱乐、AI 对话等多种功能。

## 指令分类

| 分类 | 说明 |
|------|------|
| [群管理](./group-admin) | 禁言、踢人、警告、头衔等管理功能 |
| [消息管理](./message) | 消息发送、撤回、防撤回、精华消息等 |
| [AI 功能](./ai) | AI 对话、翻译等智能功能 |
| [娱乐功能](./entertainment) | 娶群友、运势、骰子、表情包等 |
| [媒体功能](./media) | 点歌、截图、二维码等 |
| [系统管理](./system) | 插件、配置、状态、日志等 |
| [用户功能](./user) | 昵称、称呼、反馈等个人功能 |

## 快速导航

### 群管理指令
- `admin` / `unadmin` - 设置/取消管理员
- `ban` / `unban` - 禁言/解除禁言
- `kick` - 踢出用户
- `warn` - 警告用户
- `title` - 群头衔管理

### 娱乐指令
- `waifu` - 娶群友
- `jrys` - 今日运势
- `dice` - 掷骰子
- `chess` - 棋类游戏

### AI 指令
- `ai` - 与 AI 对话
- `translate` - AI 翻译

### 系统指令
- `plugin` - 插件管理
- `config` - 配置管理
- `status-a` - 查看运行状态
