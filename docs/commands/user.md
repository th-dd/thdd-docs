# 用户功能指令

用户功能指令提供个人信息管理、反馈等与用户相关的功能。

## 个人设置

### callme
修改自己的称呼，设置机器人对你的称呼方式。

**使用方式：**
```
callme [新称呼]
```

### nickname
设置用户昵称，修改群内显示名称。

---

## 反馈功能

### feedback
发送反馈信息给作者，提交建议或报告问题。

**使用方式：**
```
feedback [反馈内容]
```

---

## 帮助功能

### help
显示帮助信息，查看指令列表和使用说明。

### help-a
显示帮助信息（管理员版），查看管理员相关指令。

---

## 其他功能

### qtool
QQ 工具，提供 QQ 相关的实用功能。

### welcome
入群欢迎语管理，设置新成员入群时的欢迎消息。

### goodbye
退群欢送语管理，设置成员退群时的欢送消息。
