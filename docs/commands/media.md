# 媒体功能指令

媒体功能指令提供音乐、视频、图片等多媒体内容的搜索和生成功能。

## 音乐功能

### 点歌
搜索歌曲并播放网易云音乐。

**使用方式：**
```
点歌 [歌曲名称]
```

### 音乐搜索
搜索音乐，查找指定歌曲信息。

---

## 视频功能

### b站点播
点播 B 站视频，发送 B 站视频链接或信息。

---

## 图片功能

### shot
网页截图，将指定网页截取为图片。

**使用方式：**
```
shot [网址]
```

### qrcode
生成二维码，将文本或链接转换为二维码图片。

**使用方式：**
```
qrcode [内容]
```

### emoji
表情包功能，搜索和发送表情包。

---

## 搜索功能

### 小说搜索
搜索小说，查找网络小说资源。

---

## 其他媒体

### memes-api
制作各种沙雕表情包，生成趣味图片内容。
