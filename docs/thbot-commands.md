# 叹号Bot 指令列表

叹号Bot (以大帝结尾的bot，如句号大帝) 指令列表，提供了群管理、娱乐等多种功能。

---

::: waring

部分指令需要权限才能使用

:::

## 群管理指令

### 管理员管理

| 指令 | 说明 |
|------|------|
| `admin` | 设置管理员 |
| `unadmin` | 取消管理员 |
| `gauth` | 管理用户角色 |
| `getauth` | 获取指定成员状态 |

### 禁言管理

| 指令 | 说明 |
|------|------|
| `ban` | 禁言用户 |
| `unban` | 解除用户禁言 |
| `ban-all` | 全体禁言 |
| `unban-all` | 解除全体禁言 |
| `ban-list` | 查询当前禁言名单 |
| `banme` | 随机禁言自己 |
| `unban-allppl` | 解除所有人禁言 |
| `unban-random` | 随机解除若干人禁言 |
| `stop` | 短期禁言 |
| `forbidden` | 禁言关键词管理 |

### 用户管理

| 指令 | 说明 |
|------|------|
| `kick` | 踢出用户 |
| `warn` | 警告用户 |
| `nickname` | 设置用户昵称 |
| `title` | 群头衔管理 |
| `user` | 用户管理 |
| `verify` | 入群验证关键词管理 |

### 其他管理

| 指令 | 说明 |
|------|------|
| `inspect` | 查看用户、频道或群信息 |
| `channel` | 频道管理 |
| `grouphelper` | 群管理帮助 |
| `quit-group` | 退出指定群聊 |
