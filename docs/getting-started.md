## 什么是叹号 Bot？

叹号 Bot 是一款功能强大的 QQ 群管理机器人，提供以下核心功能：

| 功能模块 | 描述 |
|---------|------|
| 🛡️ 群管理 | 群成员管理、权限控制、自动审核 |
| 📢 消息管理 | 消息推送、定时发送、关键词回复 |
| 🎮 娱乐功能 | 签到、抽卡、小游戏 |
| 🎵 媒体功能 | 点歌、图片搜索、表情包 |
| ⚙️ 系统管理 | 群设置、Bot 配置 |

## 快速开始

### 第一步：邀请 Bot

1. 联系管理员获取 Bot 邀请链接
2. 将 Bot 邀请到你的 QQ 群
3. Bot 会自动发送欢迎消息

### 第二步：查看帮助

在群内发送以下指令查看帮助：

```
#帮助
```

### 第三步：开始使用

尝试发送一些基础指令：

```bash
#签到        # 每日签到获取积分
#状态        # 查看 Bot 运行状态
#群信息      # 查看当前群信息
```

## 基本使用

### 指令格式

叹号 Bot 的指令以 `#` 开头，例如：

```bash
#指令名 [参数]
```

::: tip 提示
部分指令需要管理员权限才能使用
:::

### 常用指令

#### 📊 查询类

| 指令 | 说明 |
|------|------|
| `#帮助` | 查看帮助菜单 |
| `#状态` | 查看 Bot 状态 |
| `#群信息` | 查看群信息 |
| `#用户信息 @用户` | 查看用户信息 |

#### 🎮 娱乐类

| 指令 | 说明 |
|------|------|
| `#签到` | 每日签到 |
| `#积分` | 查看积分 |
| `#排行` | 查看排行榜 |

#### ⚙️ 管理类（需要权限）

| 指令 | 说明 |
|------|------|
| `#设置` | 打开设置面板 |
| `#禁言 @用户 时长` | 禁言用户 |
| `#解禁 @用户` | 解除禁言 |
