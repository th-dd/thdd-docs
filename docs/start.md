## 什么是叹号 Bot？

叹号Bot 是叹号大帝旗下的机器人合集，分多个机器人运营，如果没有特殊说明，则 叹号Bot 所有权均归叹号

### 目前属于我们的机器人：

| Bot名称 | 描述 | 指令列表 | 
| :--- | :--- | :--- |
