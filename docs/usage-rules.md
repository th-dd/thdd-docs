# ⚠️ 机器人使用规范

在使用叹号旗下机器人之前，请务必仔细阅读并遵守以下规范。违反规范可能会导致机器人服务被限制或终止。

## 🚫 禁止行为

### 1. 禁止随意拉群

**未经授权，禁止将机器人拉入任何群聊。**

- 机器人仅供授权群聊使用
- 如需在群聊中使用机器人，请提前联系管理员申请
- 未经授权拉群可能会导致机器人自动退出，并记录违规行为

::: danger 违规后果
- 首次违规：机器人自动退出群聊，并收到警告
- 多次违规：可能会被列入黑名单，禁止使用所有机器人服务
:::

### 2. 禁止对 Lar 机器人发情

**请保持文明用语，禁止对 Lar 机器人进行任何形式的骚扰或不当言论。**

Lar 机器人是一个功能型助手，不是用来满足不当需求的工具。请尊重机器人服务，保持良好的使用环境。

::: warning 注意
- 对 Lar 机器人发送不当内容会被记录
- 严重违规者将被禁止使用 Lar 机器人服务
:::

### 3. 其他禁止行为

- 禁止利用机器人进行违法活动
- 禁止恶意刷指令、攻击机器人服务
- 禁止利用机器人漏洞谋取不当利益
- 禁止冒充机器人管理员或官方人员

## ✅ 正确使用方式

### 如何申请机器人入群

1. 联系管理员（叹号大帝）
2. 说明群聊用途和规模
3. 等待审核通过
4. 获得授权后方可拉入机器人
